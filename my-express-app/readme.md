โปรแกรมใช้ Express สร้าง REST API สำหรับจัดการข้อมูลนักศึกษา

มี middleware แปลงข้อมูล JSON และตรวจสอบ name กับ age ก่อนใช้งาน

รองรับ GET, POST, PUT, DELETE เพื่อดู เพิ่ม แก้ไข และลบข้อมูลนักศึกษา

ข้อมูลนักศึกษาถูกเก็บในตัวแปร students แบบ array

ใช้ HTTP status code แจ้งผลลัพธ์เมื่อสำเร็จหรือเกิดข้อผิดพลาด
